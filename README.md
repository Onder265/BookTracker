# BookTracker

**Автор:** Захаров Степан Михайлович

## Описание программы

Book Tracker — это простое desktop-приложение на Python с графическим интерфейсом (tkinter), которое позволяет:

- Добавлять книги с указанием названия, автора, жанра и количества страниц.
- Просматривать список всех добавленных книг в таблице.
- Фильтровать книги:
  - По жанру (частичное совпадение, без учёта регистра)
  - По количеству страниц (например, показать книги с числом страниц > N)
- Удалять выбранные книги.
- Сохранять все данные в JSON-файл и загружать из него.
- Проверять корректность ввода: поля не пустые, страницы — положительное целое число.
